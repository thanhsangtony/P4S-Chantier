Planification Chantier — Configuration initiale
Cette application utilise Supabase comme base de données partagée. Tous les membres de votre équipe qui ouvrent cette page verront et modifieront le même échéancier en temps réel. La configuration est à faire une seule fois.
1 Créer un compte Supabase gratuit

Allez sur supabase.com → "Start your project" → créez un compte gratuit → créez un nouveau projet (choisissez la région Canada (Central)).

2 Créer les tables

Dans votre projet Supabase, allez dans SQL Editor et exécutez ce script :

create table p4s_state (id text primary key, data jsonb, updated_at timestamptz default now());
alter table p4s_state enable row level security;
create policy "Public access" on p4s_state for all using (true) with check (true);
3 Entrez vos clés Supabase

Dans votre projet Supabase → Settings → API — copiez l'URL du projet et la clé anon public.
